# Templates

Reusable presets for jobs, gangs and public interactions.

## What is a template?

A template is a saved entity (or a saved interaction) that you can clone
into a new entry, anywhere, anytime. Use them to:

- Ship a "starter" set of jobs to friends running the same RP setting.
- Quickly recreate a job you deleted.
- Standardise the same delivery route across multiple jobs.

## Built-in templates

The script ships with a handful of starter templates: a basic medic clinic,
a public stash, a clothing store, … Open the **Templates** tab to browse.

## Saving your own

Inside the entity editor, **⋯ → Sauvegarder comme template**. Pick a name
and (optionally) a category.

## Importing into another server

Each template is a single JSON blob. Copy it from the Templates tab on
server A and paste it into the **+ Importer** dialog on server B.

This is the recommended way to share content between your own servers — no
SQL dump required.
